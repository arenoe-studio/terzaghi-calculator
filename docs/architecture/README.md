# Architecture Documentation

System design and architecture documentation for Terzaghi Calculator.

## Files

- **ARCHITECTURE.md** - System architecture overview and technical design
- **ARCHITECTURE_VISUAL.md** - Visual architecture diagrams and data flows
- **PRD.md** - Product Requirements Document
- **PROJECT_STRUCTURE.md** - Detailed project structure and file organization

## Purpose

This folder contains high-level design documentation that explains:

- How the system is structured
- Component interactions
- Data flows
- Technical architecture decisions

# Documentation

Folder ini berisi semua dokumentasi proyek Terzaghi Calculator.

## Files

### Design & Planning

- **PRD.md** - Product Requirements Document (requirements, user stories, metrics)
- **ARCHITECTURE.md** - System architecture, component design, security model
- **PROJECT_STRUCTURE.md** - Folder structure dan module organization

### Deployment & Usage

- **DEPLOYMENT_GUIDE.md** - Step-by-step deployment ke GitHub Pages + Apps Script
- **USER_GUIDE.md** - Panduan untuk end-users menggunakan calculator
- **SECURITY.md** - Penjelasan keamanan & privacy untuk users

### Technical Reference

- **API_REFERENCE.md** - Backend API documentation (request/response format)

# Assets

Static assets untuk Terzaghi Calculator.

## Folder Structure

```
assets/
├── images/           # Logo, icons, diagrams
│   ├── logo-its.png
│   ├── logo-hmds.png
│   ├── diagram-mat.png
│   └── diagram-pondasi.png
│
└── screenshots/      # UI screenshots untuk dokumentasi
    ├── calculator-ui.png
    ├── login-flow.png
    └── history-modal.png
```

## Images

### Logos

- **logo-its.png** - Logo Institut Teknologi Sepuluh Nopember
- **logo-hmds.png** - Logo Himpunan Mahasiswa Departemen Sipil

### Diagrams

- **diagram-mat.png** - Diagram pengaruh muka air tanah
- **diagram-pondasi.png** - Diagram struktur pondasi dangkal

**Cara pakai di HTML**:

```html
<img src="assets/images/logo-its.png" alt="Logo ITS" />
```

## Screenshots

Screenshots digunakan untuk:

- README.md (showcase UI)
- Documentation (tutorial visual)
- Presentations

**Format yang disarankan**:

- PNG untuk UI screenshots (lossless)
- Resolusi: 1920x1080 atau sesuai actual size
- Naming: kebab-case (lowercase dengan dash)

## Adding New Assets

1. Simpan file di folder yang sesuai
2. Gunakan naming yang descriptive
3. Optimize ukuran file (compress images)
4. Update README ini jika perlu
